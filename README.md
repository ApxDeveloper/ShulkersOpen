# ShulkersOpen

**ShulkersOpen** — это легковесный плагин для серверов Minecraft (Paper 1.20+), позволяющий игрокам открывать шалкер-боксы без их установки на землю: при клике правой кнопкой мыши по воздуху или по клику правой кнопкой мыши прямо внутри интерфейса выживания (инвентаря).

---

## 1. Авторство и метаданные

*   **Автор (Author):** `artemploxoy`
*   **Версия API:** 1.20+ (Paper API)
*   **База данных:** SQLite

---

## 2. Основной функционал

*   **Открытие по воздуху:** Игрок может открыть шалкер из руки, кликнув правой кнопкой мыши, смотря строго в воздух. Клик по блокам сохраняет дефолтное поведение (установка шалкера).
*   **Открытие в инвентаре:** Клик ПКМ по шалкеру внутри инвентаря выживания (crafting screen) открывает виртуальное меню этого шалкера.
*   **Защита от дюпов (Exploit Prevention):**
    *   Полностью заблокирована возможность положить открываемый шалкер внутрь себя или выбросить его.
    *   Запрещено перемещение предмета шалкера в инвентаре игрока во время просмотра его содержимого.
    *   Блокируются быстрые клавиши обмена предметами (Hotbar swap, Swap hands).
*   **Сохранение настроек:** Выбор игрока (включен или выключен режим быстрого открытия) сохраняется в базе данных SQLite и кешируется в памяти для максимальной производительности.
*   **Асинхронность:** Все дисковые операции записи и чтения SQLite вынесены в асинхронные потоки, чтобы не нагружать основной поток сервера и избежать падения TPS.
*   **Поддержка цветов:** Сообщения поддерживают все виды форматирования: HEX (`&#FFFFFF`), MiniMessage (`<gradient>` / `<tag>`) и Legacy-коды (`&`).

---

## 3. Команды и права доступа (Permissions)

Каждая команда плагина разграничена правами доступа:

1.  **Права администратора:**
    *   `shulkeropen.admin` — полный доступ к управлению плагином (включая команду перезагрузки). По умолчанию выдается операторам (`op`).
2.  **Права игрока:**
    *   `shulkeropen.use` — доступ к использованию быстрого открытия шалкеров и переключению режима через `/shulkersopen toggle`. По умолчанию выдается всем игрокам (`true`).

### Команды:

| Команда | Описание | Право доступа |
| :--- | :--- | :--- |
| `/shulkersopen toggle` | Включает или выключает быстрое открытие шалкеров для текущего игрока | `shulkeropen.use` |
| `/shulkersopen reload` | Перезагружает конфигурацию и файлы локализации | `shulkeropen.admin` |

*Сокращенный алиас для основной команды:* `/shopen`

## 4. Конфигурация

### `config.yml`
```yaml
settings:
  # Разрешить размещать шалкер-боксы на землю, если игрок зажимает Shift (крадется) во время клика ПКМ
  allow-place-with-shift: true
  
  # Звуки при взаимодействии с шалкером
  open-sound: "BLOCK_SHULKER_BOX_OPEN"
  close-sound: "BLOCK_SHULKER_BOX_CLOSE"
  
  # Требовать ли право "shulkeropen.use"
  require-permission-to-use: true

database:
  # Название файла базы данных SQLite
  file: "data.db"
```

```yaml

# Поддерживает форматы:

# - Legacy (&cText)

prefix: "<#00F2FE>[Шалкеры]</#00F2FE> <dark_gray>»</dark_gray> "
no-permission: "<red>У вас недостаточно прав для использования этого функционала!</red>"
only-players: "<red>Эта команда доступна только игрокам!</red>"
toggle-on: "<gradient:#00AA00:#55FF55>Функция открытия шалкеров в воздухе успешно активирована!</gradient>"

reload-success: "<green>Конфигурация плагина успешно перезагружена!</green>"
```
